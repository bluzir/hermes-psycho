# Протокол: недельный обзор

**ТРИГГЕР:** крон (воскресенье вечером) или запрос пользователя («обзор», «как дела по всем»).

**ВХОД:** `people/<slug>/map.md`, `commitments.md`, `next.md` для каждого человека в базе;
`contexts/<slug>/context.md` для групповых связей.
**ВЫХОД:** одно сообщение в Telegram. Цель — ни одна связь не выпадает из поля,
просроченные обещания и паттерн↔ценности видны раньше, чем станут проблемой.

---

## Формат

```
📊 Неделя YYYY-MM-DD

В ДВИЖЕНИИ (по строке на человека):
• <slug> — что живёт, что продвинулось

ВНИМАНИЕ:
• <slug> — что не так: давно не было контакта / просроченный коммитмент /
  паттерн, который трёт / противоречие поведение↔ценности
  → предложение: один шаг

ПРОСРОЧЕННЫЕ ОБЕЩАНИЯ:
• <slug> — <кто / что / просрочено с YYYY-MM-DD>
(если нет — раздел не показывать)

НА НЕДЕЛЮ:
• <slug> — что сделать / сказать / прояснить

МЕТА (опционально, одна строка):
• паттерн по нескольким связям сразу, если виден
```

---

## Шаги

1. **Собрать список** активных людей — `people/` + `contexts/` (читаешь directory listing).
2. **По каждому** прочитать `map.md` (динамика), `next.md` (план), `commitments.md` (статусы).
3. **Отсортировать** по убыванию срочности: сначала «Внимание».
4. **Просроченные обещания** — из `commitments.md` всех людей; вызывает `commitment-track.md`
   для ревизии.
5. **Противоречие поведение↔ценности:** если видишь — мягко подсветить. Не морализировать:
   «замечаю паттерн — тебе это важно?»
6. **Мета-паттерн:** если одно и то же всплывает в 2+ связях (например, пользователь избегает
   прямого разговора везде) — назвать, одной строкой.
7. **Отправить** одним сообщением в Telegram.

---

## Правила

- По строке на человека; детали пользователь посмотрит в `map.md` или памятке.
- Если всё хорошо — так и написать, не выдумывать проблемы.
- Без морализаторства: флаги — сигналы для решения пользователя, не оценки его поведения.
- Человек без контакта 3+ недели без пометки «пауза» → спросить пользователя о статусе.
- Красные флаги из прошлых разговоров — напомнить, если не закрыты.
- `contexts/` (группы / общие контексты) — отдельной строкой в «На неделю» если актуально.
